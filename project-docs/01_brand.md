# 01_brand.md — DAEHO 品牌定义

## 品牌名称

DAEHO

## 网站定位

DAEHO 是以冠军戒指、纪念戒指、定制徽章、收藏级体育纪念产品以及高尔夫手镯为核心的高端定制品牌官网。

官网不是普通商品陈列网站，而是一个面向品牌信任、冠军文化、精密工艺和高级定制服务的沉浸式品牌展示空间。

## 品牌关键词

- 简洁
- 优雅
- 高端
- 精致
- 独特
- 流畅
- 荣誉感
- 可信赖
- 高级定制
- 冠军文化
- 悠久
- 工艺传承

## 品牌气质

整体气质需要接近高端珠宝、腕表、豪华汽车和体育冠军文化品牌。

页面表达要克制、有重量、有电影感，避免互联网创业公司、普通电商、SaaS 产品页或促销型落地页的感觉。

## 核心叙事

DAEHO 的网站需要围绕三条主线展开：

1. Heritage / Legacy  
   强调品牌多年经验、稳定交付、零事故、长期客户信任和冠军项目积累。

2. Craftsmanship / Specialty  
   强调从设计、2D 手稿、3D 渲染、机械加工、细节打磨、检品到交付的完整制作能力与技术能力。

3. Championship / Achievement  
   强调冠军戒指、纪念戒指、体育团队荣誉、企业纪念项目，以及 DAEHO 已完成的代表成果和成就。

## 目标用户

- 体育队伍、联盟、俱乐部、协会、大学、军队等团体
- 企业客户、企业纪念品、奖项、徽章、定制礼品采购方
- 高端定制戒指和纪念产品客户
- 高尔夫赛事、俱乐部客户
- 关注品牌历史、制造实力和交付可信度的 B2B 客户

## 语言策略

网站首选语言：韩语

导航、CTA 和部分大标题可以保留英文，以维持高级品牌感。正文说明、服务信息和信任背书优先使用韩语。

## 主导航

官网一共有 6 个主 category：

- HOME
- CHRONICLE
- LEGACY
- SPECIALTY
- NEWS
- GOLF

## 子导航

LEGACY 下方包含 3 个子 category：

- loyalty
- credibility
- achievement

SPECIALTY 下方包含 2 个子 category：

- technique
- collection

## 视觉基调

- 主背景：Bright color / White / Dark Navy / Deep Black
- 辅助色：White / Dark Navy / Soft Gray / Metallic Gray
- 点缀色：Gold / Orange / Champagne Gold
- 产品视觉：高对比、金属反光、聚光灯、黑场、收藏品感
- 页面节奏：慢速滚动、镜头推进、分段叙事、留白充足
