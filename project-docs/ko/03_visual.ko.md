## 전체 비주얼 방향

DAEHO 사이트는 일반 기업 홈페이지가 아니다.

전체 방향:

- Luxury
- Cinematic
- Editorial
- Premium Craftsmanship
- Heritage
- Championship Culture
- Modern Minimalism

사이트는 고급 시계 브랜드, 럭셔리 주얼리 브랜드, 하이엔드 자동차 웹사이트, 프리미엄 스포츠 campaign에 가까워야 한다.

## 피해야 할 방향

- 스타트업 랜딩페이지
- SaaS dashboard
- 저렴한 이커머스 UI
- 과도한 테크 감성
- 밝고 가벼운 프로모션 페이지

## Layout System

권장 레이아웃:

- editorial layout
- asymmetric layout
- split layout
- showcase layout
- scroll storytelling

각 section은 하나의 브랜드 포스터처럼 보여야 한다.

## Typography

- 제목은 크고 간결하게 사용한다.
- 영어 uppercase를 적극 활용할 수 있다.
- 본문은 짧고 밀도 낮게 유지한다.
- navigation은 얇고 작게, 넓은 letter spacing을 사용한다.
- Hero headline은 영화 포스터처럼 강한 존재감을 가져야 한다.

## Color

주요 배경:

- Deep Black
- Charcoal Black
- Dark Navy Black

보조 배경:

- Metallic Gray
- Luxury Gray
- Soft Dark Gradient

포인트:

- Gold
- Champagne Gold
- Muted Blue

텍스트:

- White
- Soft Gray
- Muted Gold Accent

Gold는 포인트로만 사용한다. 넓은 면적에 밝은 금색을 깔면 안 된다.

## Image / Video Direction

제품 이미지는 고대비, 금속 반사, spotlight, premium lighting을 사용한다.

HOME Hero는 최종적으로 영상 배경을 사용한다. 현재 blueprint 이미지는 poster / placeholder다.

Hero video direction:

- championship ring blueprint
- technical drawing
- dark grid
- metallic line art
- subtle camera push
- very slow motion

영상은 제목과 navigation보다 강해지면 안 된다.

## HOME Opening Rule

순서:

1. dark intro screen
2. 중앙 `DAEHO` logo blur
3. logo sharp
4. logo가 축소와 이동을 동시에 수행
5. Hero 상단 중앙 `DAEHO` 위치에 안착
6. top bar와 Hero content reveal

## HOME Hero Top Bar

- `DAEHO`는 상단 중앙에 배치한다.
- navigation은 로고 아래에 배치한다.
- 좌측 상단은 `대호` / `OH` / `VULCAN` site links다.
- 우측 상단은 LANGUAGE / LOGIN outline pill button이다.
- top bar는 blueprint 영상 위에 overlay된다.
- 두꺼운 solid header background를 사용하지 않는다.

## Hero to Featured Categories Transition

HOME Hero에서 Featured Categories로 내려갈 때:

- Hero를 pin한다.
- 다음 화면이 아래에서 위로 밀려 올라온다.
- 다음 화면은 Hero를 덮는다.
- Featured Categories는 큰 poster / canvas처럼 보여야 한다.
- 진입 시 큰 radius, soft shadow, glow, slight scale-up을 사용할 수 있다.
- Hero는 뒤에서 살짝 어두워진다.

## Responsive

Desktop이 메인 경험이다.

Tablet:

- 복잡한 가로 구성을 2열로 단순화한다.
- heavy motion은 줄인다.

Mobile:

- section은 세로로 쌓는다.
- Hero의 시각적 임팩트는 유지한다.
- pinned transition은 단순 slide-up reveal로 대체할 수 있다.
- 이미지 / 영상 비율은 유지한다.
