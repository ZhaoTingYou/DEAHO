## HOME 확정 사항

- HOME은 5개 연속 화면 / 섹션으로 구성한다.
- 최상단은 opening animation이다.
- 중앙 `DAEHO` logo가 blur에서 sharp로 전환된다.
- logo는 선명해진 후 축소와 이동을 동시에 수행하여 Hero 상단 중앙 logo 위치로 이동한다.
- Hero top bar는 `DAEHO` 중앙, nav 하단, 좌측 site links, 우측 LANGUAGE / LOGIN 구조다.
- `대호` / `OH` / `VULCAN`은 세 개의 사이트 링크다.
- Hero background는 최종적으로 video다.
- 현재 blueprint image는 poster / placeholder다.
- Hero에서 Featured Categories로 내려갈 때 reference video처럼 pinned scroll transition을 사용한다.
